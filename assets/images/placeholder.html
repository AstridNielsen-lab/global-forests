Imagens de Placeholder para GlobalForests Pro

Este arquivo contém informações sobre as imagens necessárias para o site.

Imagens Necessárias:

1. Vídeo Hero (hero-video.mp4)

Vídeo de fundo para a seção hero. Sugestões:

  • Floresta sendo plantada
  • Tecnologia verde em ação
  • Paineis solares e árvores
  • Time-lapse de reflorestamento

2. Certificações

  • iso-14001.png - Logo ISO 14001
  • b-corp.png - Logo B-Corporation
  • carbon-trust.png - Logo Carbon Trust

3. Favicon (favicon.ico)

Icon de uma folha ou árvore em verde

Como obter as imagens:

Opção 1: Bancos de Imagens Gratuitos

  • Unsplash: https://unsplash.com/ (busque por "forest", "sustainability", "green technology")
  • Pexels: https://pexels.com/ (alta qualidade, gratuitas)
  • Pixabay: https://pixabay.com/ (várias opções gratuitas)

Opção 2: IA Gerativa

  • Midjourney: Para imagens de alta qualidade
  • DALL-E: Para criações personalizadas
  • Stable Diffusion: Alternativa open-source

Opção 3: Vídeos

  • Pexels Videos: https://pexels.com/videos/
  • Coverr: https://coverr.co/
  • Videvo: https://videvo.net/

Especificações Técnicas:

Vídeo Hero:

  • Formato: MP4 (H.264)
  • Resolução: 1920x1080 (Full HD) ou 4K
  • Duração: 10-30 segundos (loop)
  • Tamanho: Máximo 10MB (para web)
  • Sem áudio

Imagens:

  • Formato: JPG (fotos), PNG (logos com transparência)
  • Resolução: Mínimo 1920px de largura
  • Qualidade: Alta para hero, otimizada para web
  • Tamanho: Máximo 1MB por imagem

Favicon:

  • Formato: ICO, PNG
  • Tamanhos: 16x16, 32x32, 48x48 pixels
  • Simples e reconhecível em tamanho pequeno
